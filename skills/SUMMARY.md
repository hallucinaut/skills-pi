# Skills Collection

This directory contains custom skills for the Pi coding agent.

## Available Skills

| Skill | Description | Type |
|-------|-------------|------|
| backend-api | Build RESTful APIs, GraphQL endpoints, and database-driven backend services | Backend |
| code-review | Analyze code quality, security vulnerabilities, performance issues, and best practices | Code Quality |
| database-schema | Design and document database schemas, migrations, relationships, and constraints | Database |
| frontend-design | Design and implement distinctive, production-ready frontend interfaces | Frontend |
| security-audit | Perform security assessments, vulnerability scanning, and penetration testing | Security |
| testing | Write unit tests, integration tests, and end-to-end tests | Testing |
| docker | Containerize applications with Docker and Docker Compose | DevOps |
| cloud | Manage cloud resources across AWS, GCP, or Azure | Cloud |
| kubernetes | Deploy and manage Kubernetes applications | DevOps |
| monitoring | Set up monitoring, logging, and observability | DevOps |
| devops | Set up CI/CD pipelines, infrastructure as code, and deployment workflows | DevOps |
| data-analysis | Analyze data, create visualizations, and generate insights | Data |
| debugging | Diagnose and fix bugs efficiently | Debugging |
| documentation | Write and maintain clear, comprehensive documentation | Documentation |

## Installation

Add this directory to Pi's skills configuration:

```json
{
  "skills": ["~/.pi/skills"]
}
```

Or use the repository:

```bash
git clone https://github.com/hallucinaut/skills-pi.git
```
